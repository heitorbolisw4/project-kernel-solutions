# Kernel Solutions — Portfólio

Landing page institucional para apresentar a Kernel Solutions, destacar serviços de tecnologia e captar contatos de clientes potenciais.

## Visão geral
- **Stack:** HTML, CSS e JavaScript puro.
- **Estrutura:** seção hero, prova social/dor, módulos de serviços, grid de portfólio e formulário multietapas.
- **Responsividade:** layout fluido com breakpoints otimizados para desktop, tablet e mobile (hambúrguer menu, grid adaptativo e tipografia responsiva).

## Como executar
1. Clone ou baixe o repositório.
2. Abra `index.html` em qualquer navegador moderno. Não há dependências externas ou build.

## Experiência do usuário
- Formulário guiado com validações de entrada e mensagens de erro acessíveis.
- Navegação mobile com abertura/fechamento pelo botão hambúrguer e pelo teclado (Esc).
- Seções com espaçamento padronizado para manter ritmo visual entre blocos de conteúdo.

## Estrutura das pastas
- `assets/` – imagens usadas no layout.
- `styles/` – folhas de estilo da página.
- `script/` – lógica do menu responsivo e do formulário interativo.
- `index.html` – marcação principal da landing page.
